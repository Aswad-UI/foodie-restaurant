# Foodie Restaurant Website 🍽️

A responsive, modern restaurant website built with **HTML, CSS, and JavaScript**.  

This project demonstrates front-end web development skills including layout design, responsiveness, and interactivity.

---

## 🔗 Live Demo
[View Live Website](https://Aswad-UI.github.io/foodie-restaurant/)

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Git & GitHub (for version control)
- Responsive Design (mobile-friendly)

---

## 📂 Features
- Fully responsive design for desktop, tablet, and mobile
- Menu section with detailed food items
- About, Gallery, and Contact sections
- Smooth scrolling navigation
- Interactive elements like hover effects
